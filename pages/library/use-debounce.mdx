# Use Debounce

</Header>

Utilitas untuk menambahkan teknik debouncing pada react.

Debouncing adalah teknik untuk mengurangi suatu event terjadi berulang kali. contohnya pada saat typing pada input search, fungsi search akan berjalan setiap ada perubahan karakter, nah debouncing dapat mengurangi fungsi search agar tidak berjalan pada setiap perubahan karakter.

[Lihat Dokumentasi](https://www.npmjs.com/package/use-debounce)
